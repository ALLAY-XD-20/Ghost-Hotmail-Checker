# 👻 Ghost Hotmail Checker (Termux Edition)

A **terminal-based Hotmail / Outlook account checker** designed to run **100% inside Termux** using **Textual**.

This project was originally GUI-based (`tkinter / customtkinter`) and has been **fully upgraded and fixed for Termux**.

> ✅ No GUI  
> ✅ No tkinter  
> ✅ Works on Android (Termux)  
> ✅ Fast & stable  

---

## ✨ Features

- 🔍 Hotmail / Outlook account checkin
- ⚡ Multi-threaded checking
- 🌐 Optional proxy support
- 🟩 Green **VALID** / 🟥 Red **INVALID** results
- 📄 Auto-save valid hits to `valid_hits.txt`
- 🔔 Discord webhook support
- 🧵 Thread-safe logging
- 📱 Fully compatible with Termux

---

## 🛠 Requirements

- Termux (latest)
- Python **3.12+**
- Internet connection
